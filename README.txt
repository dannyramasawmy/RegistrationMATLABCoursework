This coursework is submitted for Programming Foundations for Medical Image Analysis.
This coursework folder has been split into three folders, data, src and testScripts.

The aim of this coursework was to register two medical images together (MRI and US) and analyse the results for surgical planning.

./data 		:The data folder has all the data for the MRI and US for phantom and patient. (data could not be included)
./src  		:src contains all the functions needed to analyse the data. Any sub functions the src code calls have been nested.
./testScripts 	:contains the test scripts for running the functions in src.
